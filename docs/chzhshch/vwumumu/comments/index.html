000.所有批注

逻辑线​

两条长短周期的均线可以将走势分类。

均线的关系有三种：飞吻、唇吻、湿吻。

湿吻表现为均线缠绕。

缠绕分为中继和转折。

先不考虑缠绕转折的情况。

对于中继缠绕后的背驰是第一个买点，之后女上位第一次缠绕不新低是第二个买点。

越来越高或越来越低的缠绕为趋势，横盘缠绕为盘整。

在趋势中或趋势之间可以通过均线面积大小比较力度，力度变小为背驰。

高级别的趋势的一段，可能是低级别的一整个趋势，这是级别的同构性关系。
