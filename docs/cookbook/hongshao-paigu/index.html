红烧排骨

材料​

猪肋排2斤、葱、姜、桂皮、花椒、大料、生抽、冰糖老抽、盐、糖、白胡椒、热水。

做法​

焯水​

  1. 锅中放入冷水，水中放入葱段2-3个，姜片4-5片，一点黄酒。
  2. 将猪肋排放入锅中后，开火
  3. 水烧开的中途放入两个大料，水烧开后取出
  4. 水烧开后1-2分钟关火，将焯好的猪肋排用热水洗净

炒肋排​

  1. 热锅凉油，放入2-3段葱，3-4片姜，花椒10多粒，桂皮1块，大料2个爆香。
  2. 倒入热水洗净的猪肋排翻炒
  3. 倒入大概1.5炒菜勺的酱油，半炒菜勺冰糖老抽，翻炒给排骨上色，老抽的量也可以根据排骨需要上色的程度来考虑。
  4. 大概1分钟，充分炒上色后倒入热水，没过排骨，放一点点盐，大火烧开

高压锅压熟​

  1. 大火10-15分钟后，将全部食材倒入高压锅
  2. 传统高压锅压大概10-15分钟，现代电子压力锅选择猪肉档压制

调口味​

  1. 高压锅压制完成后，倒出到炒菜锅中收汁，过程中将姜，大料，桂皮等材料挑出
  2. 收汁过程中，稍微尝一下汁的口味，根据咸度，可以加一点盐，为了提鲜，加一点白糖

收汁​

  1. 将排骨挑出，单独收汁，待汤汁量合适，将排骨倒入锅中颠几下出锅成盘。

注意事项​

无
